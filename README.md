# dotfiles-linux

## Requirements

- GNOME Shell 46.0
___
## Apps via Snap

- Obsidian
___
## Extensions via apt

- gnome-shell-extension-manager
- rclone (Google Drive)
___
# Commands
- Mount local Google Drive:
```commandline
    rclone mount gdrive: ~/GoogleDrive --vfs-cache-mode writes
```
___
